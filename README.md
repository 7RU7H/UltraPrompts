# UltraPrompts


Free Open Source Prompt Templates for everyone 

Combine with UltraPrompt-AIQC for easier copy pasting on Linux 

Diane for example
```bash
./UltraPrompts-AIQC -p $DIR/UltraPrompts/What-Would-Project-SHODAN-Diane-Say.md -q "" | xsel -b
```
